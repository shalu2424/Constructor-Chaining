🔗 Constructor Chaining in Java

Constructor Chaining is a concept in Object-Oriented Programming (OOP) where one constructor calls another constructor of the same class or parent class.

It helps to reuse constructor code and reduce duplication.

📌 Ways to Perform Constructor Chaining
1️⃣ this()

Used to call another constructor of the same class.

Must be the first statement inside the constructor.

Only one this() call is allowed in a constructor.

2️⃣ super()

Used to call the constructor of the parent class.

Also must be the first statement in the constructor

🎯 Key Points
✔ Constructor chaining improves code reusability
✔ Reduces duplicate initialization code
✔ this() calls constructor of same class
✔ super() calls constructor of parent class
